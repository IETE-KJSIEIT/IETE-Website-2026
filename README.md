# IETE KJSIT – Official Technical Hub

> **Where Innovation Converges**

The official website of the IETE Student Chapter at K. J. Somaiya Institute of Technology (KJSIT), showcasing our events, team, and technical initiatives.

---

## 🚀 About IETE KJSIT

The IETE Student Chapter at KJSIT is a platform for innovators, developers, and problem-solvers. We aim to bridge the gap between theoretical learning and practical implementation through events, workshops, and collaborative projects, fostering a strong technical community.

## 📁 Project Structure

```text
iete-oscillation/
├── public/                 # Static assets (images, logos, icons)
│   ├── logos/
│   ├── sponsors/
│   └── team/
├── src/                    # Source code
│   ├── assets/             # Project assets
│   ├── components/         # Reusable React components (Navbar, Hero, Footer, etc.)
│   ├── data/               # Static data definitions (eventsData, teamData)
│   ├── hooks/              # Custom React hooks
│   ├── pages/              # Route-level pages (Home, Events, Oscillation)
│   ├── App.jsx             # Main application routing and Code Splitting
│   ├── main.jsx            # React rendering entry point
│   └── index.css           # Global stylesheet and variables
├── index.html              # Main HTML template
├── package.json            # Project dependencies and scripts
└── vite.config.js          # Vite configuration
```
